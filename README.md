# Analisi Diffusione COVID-19 - Esame finale Epicode

## Obiettivo
L’obiettivo è analizzare l’andamento dei casi e delle ospedalizzazioni dovute al COVID-19, sfruttando il dataset "Our World In Data".

## Dataset
Il dataset utilizzato è disponibile su GitHub: https://github.com/owid/covid-19-data/tree/master/public/data

## Analisi effettuate
1. **Esplorazione dati (EDA)**: verificate le dimensioni del dataset, le colonne principali e un esempio di record.
2. **Casi per continente**:
   - Totale casi per continente.
   - Percentuale sul totale mondiale.
3. **Italia 2022**:
   - Evoluzione dei casi totali.
   - Nuovi casi settimanali.
4. **Confronto terapia intensiva**: pazienti in ICU per Italia, Francia e Germania nel 2022-2023.
5. **Ospedalizzazioni 2021**: pazienti ospedalizzati in Italia, Francia, Germania, Spagna.
